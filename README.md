# hungercracy
Sistema de votação para decidir o restaurante do dia

Tecnologias utilizadas:

Java EE
Funcionalidades de data com a nova implementação do java 8 java.time.LocalDate
MVC: Spring Boot + ThymeLeaf
View: HTML JavaScript + JQuery
Repositório: Git
Micro Serviço: Spring Boot
Banco de dados: JPA + H2 database
Testes: Junit e Mockito para testes unitários e testes Spring MV para a  Web Layer.
Build: Gradle
